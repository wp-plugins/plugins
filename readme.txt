== Description ==

With this tiny plugin you can show plugins stats for any wordpress contributor from wordpress.org

== Frequently Asked Questions ==

**How can I submit a bug?**

Write me to lazarov@mail.bg, please, it's the quicker way to have it fixed. You can write to the 
WordPress forum, too, but I read it rarely.

**Other FAQs?**

I'm collection tips and FAQ on [this page](http://marto.lazarov.org/plugins/plugins/) 
so I can update it more easily.
